# PIG GAME

- ## Rules of the game : 
    - Single and Multiplayer game.
    - Player one decide the number of times he wants to roll a die.
        - If the output is 2,3,4,5,6 : it adds to their overall score.
        - If the output is 0 : their current score becomes 0.
    - The chances were go from one player to another and first to reach a specific score wins.
