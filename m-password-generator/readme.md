# Password Generator

> A customizable script that allows a user to generate secure passwords with optional numbers and special characters.

## Requirements :

- Users must provide the desired password length at runtime.
- Optional inclusion of numeric characters based on user input.
- Optional inclusion of special characters based on user input.
- The script must guarantee that if numbers or special characters are enabled, at least one of each is included in the final password.

- A clean exit occurs automatically after displaying the generated password.
