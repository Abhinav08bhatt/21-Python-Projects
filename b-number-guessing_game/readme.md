# Number guessing game

> A command-line game where the user attempts to guess a random number chosen by the computer.

### Requirements :  
- The program must ask the user to define the upper bound for the number to be guessed.
- Validate the upper bound input to ensure it is a positive integer, if not re-prompt the user.
- Use the random module to select a secret number between 1 and user's specified upper bond.
- Track the number of attempts the user takes.
- Give hints if the user's guess is above of below the selected number.
- store the high score in a separate text file (top 5)
