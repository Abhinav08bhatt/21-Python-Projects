# YouTube Video Downloader

A Python script that downloads YouTube videos in the highest available MP4 resolution and saves them to a user-selected folder using a file dialog.

## Requirements :

- Users must input a valid YouTube video URL at runtime.
- The program must allow the user to select a download folder using a GUI file dialog.
- The script must download the highest resolution available MP4 video stream.
- Filenames must be cleaned to avoid invalid characters.
- The download progress must be displayed as a percentage in the terminal.
- The program must print video details (title, author, length) before downloading.
- The program must handle errors gracefully if the download fails.
