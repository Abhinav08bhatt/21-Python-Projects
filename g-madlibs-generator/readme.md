# Madlibs Generator

A fun and interactive script that takes a story template and turns it into a fully customized using user-provided words.

# Features

- Reads a story from a template file containing placeholders like [NOUN], [ADJECTIVE], and [VERB].
- Automatically detects all unique placeholders and prompts the user only once for each.
- Replaces all placeholders and prints the final completed story.
- Supports multiple placeholder occurrences across the story.
